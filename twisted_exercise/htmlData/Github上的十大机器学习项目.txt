<script type="text/javascript">
    $(function()
    {
        //取消背景层的滚动条滚动样式
        /*$(document).css(
            {"overflow":"none","overflow":"hidden","_overflow":"hidden","padding":"0 17px 0 0"}
        );*/

        $('#accept').removeAttr('checked');
        
        //申请创建主题弹框
        $('body').delegate('.apply_CreatF', 'click', function()
        {
            var login_username = '';
            var cookie = document.cookie.split(";");
            for (var i in cookie){
                var val = cookie[i];
				var tmp = val.split('=');
				if (tmp[0].trim() == 'UserName'){
					login_username = tmp[1];
				}
            }
        	if (login_username == ''){
				location.href = "https://passport.csdn.net/account/login";
            }
        	$(".mark_apply_create").appendTo('body').removeClass("hide").fadeIn(100);
        	$(".mark_apply_create").find(".popup_c:eq(0)").show().siblings().hide();

        });

        $("body").delegate('#accept', 'click', function()
        {
            checkState();
            document.documentElement.style.cssText = 'overflow:none;+overflow:hidden;_overflow:hidden;';
            document.body.style.cssText = 'overflow:hidden;+overflow:none;_overflow:none;padding:0 17px 0 0;';
        });

        //提交创建主题
        $('body').delegate('.create_forum_submit', 'click', function(){
            var tel = $(".popup_list input[type='tel']").val();
            if($('.input_forum_name').val().trim() == ""){
                $('.create_forum_error').html('请填写主题名称').show();
                $('.input_forum_name').focus();
            }else if($('.input_forum_describe').val().trim() == ""){
                $('.create_forum_error').html('请填写主题描述').show();
                $('.input_forum_describe').focus();
            //}else if(/^13\d{9}$/g.test(tel)||(/^15[8,9]\d{8}$/g.test(tel)||(/^15[8,9]\d{8}$/g.test(tel))
            }else if(/^1[3||5||8]\d{9}$/g.test(tel))
            {
                $(this).attr("disabled", "disabled");
                var data = $('.create_forum_form').serialize();
                $.post("/forum/create", data, function(result){
                    if(result.status){
                        $('.create_forum_error').hide();
                        $(".mark_apply_create").find(".popup_c:eq(2)").show().siblings().hide();
                    } else{
                        $('.create_forum_submit').attr("disabled", false);
                        $('.create_forum_error').html(result.error).show();
                    }
                });
            }
            else
            {
                $('.create_forum_error').html('请填写正确的手机号').show();
                $(".popup_list input[type='tel']").focus();
            }
			return false;
        });
        
        //状态检查
        function checkState()
        {
            var flag = false;
            var isCheck = $("#accept").prop("checked");
            if(isCheck != null && isCheck == true)
            {
                flag = true;

            }
            if(flag)
            {
            	$('#next').removeAttr("disabled").addClass("next_02");
                $('body').delegate('#next', 'click', function()
                {
                    $(".mark_apply_create").find(".popup_c:eq(1)").show().siblings().hide();

                });
            }
            else
            {
                $('#next').attr("disabled",true).removeClass('next_02');
            }
            //alert(flag);

        };


        $("body").delegate('.close', 'click', function()
        {
        	$(".mark_apply_create").addClass("hide");
            document.documentElement.style.cssText = '';
            document.body.style.cssText = '';
        });
        
    });
</script>

				<div class="news_description ">
                        <div class="description markdown_views">
                <blockquote>
  <p>原文作者：<a href="https://twitter.com/mattmayo13">Matthew Mayo</a> <br>
  译文地址：<a href="http://www.kdnuggets.com/2015/12/top-10-machine-learning-github.html">Top 10 Machine Learning Projects on Github</a> <br>
  文章译者：<a href="http://www.cnblogs.com/naive/">赵屹华</a>，搜狗计算广告工程师，前生物医学工程师，关注推荐算法、机器学习领域。 <br>
  文章审校：刘帝伟</p>
</blockquote><pre><code>Github上的十大机器学习项目涵盖了一系列函数库、框架和教学资源。我们来看看别人使用的工具和学习的资源。
</code></pre><p>开源软件是数据科学很重要的一部分。</p><p>根据最近的<a href="http://www.kdnuggets.com/2015/05/poll-r-rapidminer-python-big-data-spark.html">KDnuggets数据科学软件投票的结果</a>，73%的数据科学家在过去12个月里使用过免费软件。互联网上有着各式各样的这类工具，而Github事实上则成为了所有开源软件的交流平台，包括数据科学社区里所用的工具。机器学习在数据科学界的重要性和中心地位已经不言而喻。</p><p>下图是<a href="https://github.com/search?o=desc&amp;q=Machine+Learning&amp;s=stars&amp;type=Repositories&amp;utf8=%E2%9C%93">Github十大机器学习项目</a>的概览。</p><p><img src="http://img.blog.csdn.net/20160121211449669" alt="图片描述" title=""></p><p><a href="https://github.com/scikit-learn/scikit-learn"><strong>1. Scikit-learn</strong></a></p><blockquote>
  <p>Python的机器学习库</p>
</blockquote><p>★8641，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">5125</p><p>十大之首，毫无悬念地就是工业界和学术界Python开发者首选的机器学习函数库。Scikit利用了Python的科学计算工具，它基于Numpy，Scipy和matplotlib。Scikit-learn拥有一般工具包的常规功能，包括分类、回归和聚类算法，也包括数据预处理和模型评价模块。</p><p><img src="http://img.blog.csdn.net/20160121211811532" alt="图片描述" title=""></p><p><a href="https://github.com/josephmisiti/awesome-machine-learning"><strong>2. Awesome Machine Learning</strong></a></p><blockquote>
  <p>一系列绝妙的机器学习框架、函数库和软件。</p>
</blockquote><p>★ 8404，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">1885</p><p>这是一系列绝妙的机器学习框架、函数库和软件。这个列表先按照语言来分类，然后按照机器学习的类别（通用型，计算机视觉，自然语言处理，等等）。它还包括数据可视化工具，从某种意义上来说它比数据科学的通用列表更丰富，这是一件好事。</p><p><a href="https://github.com/PredictionIO/PredictionIO"><strong>3. PredictionIO</strong></a></p><blockquote>
  <p>PredictionIO是开发者和ML工程师的一个机器学习服务器。它基于Apache Spark、HBase和Spray。</p>
</blockquote><p>★ 8145，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">1002</p><p><a href="https://github.com/PredictionIO/PredictionIO">PredictionIO</a>是一个通用型框架。它包括一些处理常规问题的模板引擎，比如分类和推荐，也可以用户自定义修改，通过REST APIs或者SDKs与现有的应用连接。由于它是建立在Spark基础上并且利用了Spark的生态系统，因此PredictionIO主要用Scala开发也就不足为奇了。</p><p><img src="http://img.blog.csdn.net/20160121212040493" alt="图片描述" title=""></p><p><a href="https://github.com/hangtwenty/dive-into-machine-learning"><strong>4. Dive into Machine Learning</strong></a></p><blockquote>
  <p>使用Python Jupyter和Scikit-learn深入研究机器学习。</p>
</blockquote><p>★ 4326，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">342</p><p>这是scikit-learn的一个教程集合，有大量IPython notebook脚本，和许多Python相关的或者通用型的机器学习话题的链接，以及更多的数据科学信息。作者并不很贪婪，如果某一个教程不足以吸引你，它们会很快发掘更多的其它类似内容。代码库里并没有软件，但如果你不熟悉Python环境下的机器学习，则值得读一下。</p><p><a href="https://github.com/clips/pattern"><strong>5. Pattern</strong></a></p><blockquote>
  <p>Python的网页挖掘模块，包括爬虫、自然语言处理、机器学习、网络分析和可视化等工具。</p>
</blockquote><p>★ 3799，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">598</p><p><a href="http://www.clips.ua.ac.be/pages/pattern">Pattern</a>是基于Python的网页挖掘工具包，由Antwerp大学的计算语言学和心理语言学研究中心（<a href="http://www.clips.ua.ac.be/pages/pattern">Computational Linguistics &amp; Psycholinguistics, CLiPS</a>）开发完成。它可以用来完成爬虫、机器学习、自然语言处理、网络分析和可视化等任务。Pattern还可以从一些熟知的网络服务中挖掘数据。这个项目有完整的文档，并且包含了大量的例子和单元测试。</p><p><img src="http://img.blog.csdn.net/20160121212252182" alt="图片描述" title=""></p><p><a href="https://github.com/numenta/nupic"><strong>6. NuPIC ( Numenta Platform for Intelligent Computing)</strong></a></p><blockquote>
  <p>一个大脑启发式的机器智能平台和基于大脑皮层学习算法的生物神经网络。</p>
</blockquote><p>★ 3647，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">987</p><p><a href="http://numenta.org/">NuPIC</a>实现了<a href="http://numenta.com/learn/hierarchical-temporal-memory-white-paper.html">Hierarchical Temporal Memory (HTM)</a>机器学习算法。HTM算法试图以大脑皮层的计算方式来建模，专注于存储和召回空间和时间的模式。NuPIC非常适合模式相关的异常检测。</p><p><a href="https://github.com/JohnLangford/vowpal_wabbit"><strong>7. Vowpal Wabbit</strong></a></p><blockquote>
  <p>Vowpal Wabbit是一种机器学习系统，它在online, hashing, allreduce, reductions, learning2search, active 和 interactive learning等技术上一直处于领先优势。</p>
</blockquote><p>★ 2949，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">827</p><p><a href="http://hunch.net/~vw/">Vowpal Wabbit</a>的目标是对大数据快速建模，并支持并行学习。这个项目由雅虎发起，现在由微软研究院开发。Vowpal Wabbit采用了外部存储学习算法（<a href="https://en.wikipedia.org/wiki/Out-of-core_algorithm">out-of-core</a>），它已经实现了用1000个计算节点在一小时内<a href="http://arxiv.org/abs/1110.4198">训练TB级的特征数据集</a>。</p><p><a href="https://github.com/airbnb/aerosolve"><strong>8. aerosolve</strong></a></p><blockquote>
  <p>一个交互友好的机器学习工具包</p>
</blockquote><p>★ 2538，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">245</p><p>aerosol与其它的函数包不同，它主要是提供交互友好的调试工具，训练模型的Scala代码，一个用于图像排序的图像内容分析引擎，和一种特征转换语言，用户可以灵活地控制特征。aerosolve采用基于thrift的特征表征，特征按照逻辑被分组后进行变换，或者一次性对所有特征组完成变换。</p><p><img src="http://img.blog.csdn.net/20160121212458026" alt="图片描述" title=""></p><p><a href="https://github.com/sjwhitworth/golearn"><strong>9. GoLearn</strong></a></p><blockquote>
  <p>一种Go语言的机器学习工具。</p>
</blockquote><p>★ 2334，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">215</p><p>GoLearn是使用Go语言开发的机器学习库，开发的活跃度很高。它的目的是为Go语言开发者提供一套完善的、易用的、可自定义的工具包。GoLearn实现了Scikit-learn中常用的fit/predict接口，简化了预测器的生成方法，并实现了交叉验证、训练集/测试集切分等常用函数。</p><p><a href="https://github.com/johnmyleswhite/ML_for_Hackers"><strong>10. Code for Machine Learning for Hackers</strong></a></p><blockquote>
  <p>Machine Learning for Hackers一书中的代码</p>
</blockquote><p>★ 2003，<img src="http://img.blog.csdn.net/20160121211650593" alt="图片描述" title="">1446</p><p>这个代码库中的代码都来自O’Reilly出版的<a href="http://shop.oreilly.com/product/0636920018483.do">Machine Learning for Hackers</a>一书。代码用R语言实现，其依赖了大量R工具包，它的内容包括常见的分类任务、排序和回归，以及主成分分析和多维标度法等统计方法。</p><p><strong>注：</strong>上榜依据是在Github上搜索“Machine Learning”关键词所返回的结果，按照星星的数量排序，数据搜集时间是2015年12月10日下午1点。</p><p><strong>相关阅读：</strong></p><ul>
<li><a href="http://www.kdnuggets.com/2015/06/top-20-python-machine-learning-open-source-projects.html">Top 20 Python Machine Learning Open Source Projects</a></li>
<li><a href="http://www.kdnuggets.com/2015/11/topological-data-analysis-open-source-implementations.html">Topological Data Analysis – Open Source Implementations</a></li>
